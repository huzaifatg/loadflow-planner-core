# LoadFlow Planner Core

Open-source logistics planning utilities extracted from LoadFlow.

## Overview

LoadFlow Planner Core provides reusable utilities for developers building:

* Fleet management software
* Dispatch systems
* Transportation platforms
* Logistics planning applications

## Features

* Truck capacity validation
* Weight utilization calculations
* Delivery assignment checks
* Planning helpers

## Example

```ts
const result = calculateUtilization(
  truck,
  deliveries
);
```

## Roadmap

* Volume constraints
* Pallet capacity support
* Time window validation
* Delivery sequencing helpers
* Planning heuristics

## Status

Early-stage project actively maintained alongside the LoadFlow logistics platform.
